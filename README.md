# Ananke
Just another lightweight dockerized PHP Development Framework

Getting started
---

Edit your hosts file:

```bash
127.0.0.1   ananke.local
```
Install composer dependencies:
```bash
composer install
```

Run the app:

```bash
docker-compose up
```
